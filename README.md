# 家計簿アプリ

Pythonで作成した、コマンドライン操作のシンプルな家計簿アプリです。

収入・支出の登録、一覧表示、月ごとの集計、登録データの削除ができます。データはCSVファイルに保存されるため、アプリを終了したあとも記録を残せます。

## 機能

- 収入の登録
- 支出の登録
- 登録データの一覧表示
- 月ごとの収入・支出・収支の集計
- 登録データの削除
- CSVファイルへの保存

## 使用技術

- Python
- CSV

外部ライブラリは使用していません。

## ファイル構成

```text
.
├── README.md
├── main.py
└── records.csv
```

## 起動方法

以下のコマンドでアプリを起動します。

```bash
python main.py
```

環境によっては、次のコマンドを使用してください。

```bash
python3 main.py
```

## 使い方

起動すると、以下のようなメニューが表示されます。

```text
家計簿アプリ
1. 収入を登録
2. 支出を登録
3. 一覧を表示
4. 月ごとの合計を表示
5. 登録データを削除
6. 保存して終了
```

番号を入力して、実行したい操作を選択します。

## データ形式

登録データは `records.csv` に保存されます。

```csv
date,type,category,amount,memo
2026-04-15,income,給料,141706,無印良品
```

各項目の意味は以下の通りです。

| 項目 | 内容 |
| --- | --- |
| date | 日付 |
| type | `income` または `expense` |
| category | カテゴリ |
| amount | 金額 |
| memo | メモ |

## 月ごとの集計

月ごとの合計を表示する場合は、`2026-05` のように年月を入力します。

対象月の収入合計、支出合計、収支が表示されます。
